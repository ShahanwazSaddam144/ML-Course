# 📘 Introduction to Mathematics

------------------------------------------------------------------------

## 🔹 Mean (Average Value)

**Definition:**\
The mean is the average of all values in a dataset.

**Formula:** Mean = (x₁ + x₂ + ... + xₙ) / n

------------------------------------------------------------------------

## 🔹 Median (Middle Value)

**Definition:**\
The median is the middle value after sorting the data in ascending or
descending order.

------------------------------------------------------------------------

## 🔹 Mode

**Definition:**\
The mode is the value that appears most frequently in the dataset.

------------------------------------------------------------------------

## 🧮 Example

**Given Data:**\
10, 20, 30

**Step 1: Find Sum**\
10 + 20 + 30 = 60

**Step 2: Calculate Mean**\
Mean = 60 / 3 = 20

**Results:** - Mean: 20\
- Median: 20\
- Mode: No mode (all values are unique)

------------------------------------------------------------------------

## 📊 Percentage

**Example:**\
60 runs in 100 balls

-   Ratio = 60 : 100\
-   Percentage (Strike Rate) = 60%

------------------------------------------------------------------------

## 📈 Standard Deviation

**Definition:**\
Standard deviation measures how much the data values deviate from the
mean.

**Formula:**\
σ = √(1/n \* Σ(xᵢ - μ)²)

**Where:** - xᵢ = each data point\
- μ = mean\
- n = number of values\
- σ = standard deviation

------------------------------------------------------------------------

## 🧠 Key Insight

-   Small σ → Data is close to the mean\
-   Large σ → Data is widely spread

------------------------------------------------------------------------

## ✅ Summary

  Concept   Meaning
  --------- -----------------------
  Mean      Average value
  Median    Middle value
  Mode      Most frequent value
  Std Dev   Data spread/variation
