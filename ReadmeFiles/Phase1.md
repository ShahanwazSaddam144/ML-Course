Foundation of Machine Learning (Phase-1)
----------------------------------------

Diff between AL, ML, DL
------------------------

Artifical Learning (AI):
Artificial Intelligence is the broad field of creating machines that 
can simulate human intelligence. 

Goal:
Make machines think, learn, and act like humans.

Examples:

Chatbots
Self-driving cars
Game-playing systems
--------------------------------------

Machine Learning (ML):
Machine Learning is a subset of AI that allows systems to learn from data 
without being explicitly programmed.

Key Idea:
“Learn from data and improve over time.”

Examples:

Spam email detection
Recommendation systems (Netflix, YouTube)
Fraud detection
----------------------------------------


Deep Learning (DL):
Deep Learning is a subset of ML that uses neural networks with many layers.

Key Idea:
“Learn complex patterns using large neural networks.”

Examples:

Image recognition
Speech recognition
Face detection
------------------------------------------------

Types of Machine Learning
-------------------------
🔹 1. Supervised Learning

Definition:
Model learns from labeled data (input + correct output).

Idea:
“Learn with a teacher”

Examples:

Predicting house prices
Spam email detection

Common Tasks:

Classification (Yes/No, Spam/Not Spam)
Regression (predict numbers)

-----------------------------

2. Unsupervised Learning

Definition:
Model learns from unlabeled data (no correct answers given).

Idea:
“Find patterns on your own”

Examples:

Customer segmentation
Grouping similar products

Common Tasks:

Clustering
Association
-----------------------------

3. Reinforcement Learning

Definition:
Model learns by trial and error using rewards and penalties.

Idea:
“Learn from experience”

Examples:

Game playing (chess, AI agents)
Self-driving cars
